# How to run

- Please set environment variables before run.
  |Name            |Require |Description                   |Default                              |
  |----------------|--------|------------------------------|-------------------------------------|
  |PORT            | ❌     | Port to running server       | 3001                                |
  |MONGO_URI       | ❌     | Connect string to MongoDb    | mongodb://localhost:27017/ecommerce |

- Run server by command:
  ```bash
  npm i && npm run start
  ```
